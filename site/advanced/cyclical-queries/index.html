If you have cyclical queries in your graph, you may want to limit the possible query depth.

You can use instrumentation to monitor and restrict query depth. In graphql-java, you can find an implementation for this in MaxQueryDepthInstrumentation.java.

If you register an Instrumentation class as a Spring bean, the [DGS] framework will pick it up automatically. For example:

@Bean
public Instrumentation
maxQueryDepthInstrumentation() {
   return new MaxQueryDepthInstrumentation(5);
}
